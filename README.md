# JejakSentul.id - Website Jasa Guide Trekking

Website profesional untuk layanan jasa guide trekking di Sentul, Bogor. Dibuat dengan HTML5, CSS3, dan JavaScript vanilla untuk performa optimal.

## 🏔️ Fitur Utama

- **Responsive Design** - Tampil sempurna di semua device
- **3 Tingkat Kesulitan** - Mudah, Menengah, dan Sulit
- **15+ Rute Trekking** - Curug, gunung, dan goa eksotis
- **Booking Terintegrasi** - Langsung ke WhatsApp (085163689179)
- **Galeri Foto** - Showcase keindahan destinasi
- **SEO Optimized** - Mudah ditemukan di Google

## 🚀 Cara Deploy

### GitHub Pages (Gratis)
1. Push semua file ke repository GitHub
2. Aktifkan GitHub Pages di Settings
3. Pilih branch main
4. Website akan tersedia di: `https://username.github.io/repository-name`

### Custom Domain
1. Beli domain jejaksentul.id
2. Tambahkan file `CNAME` dengan isi domain
3. Setting DNS A record ke IP GitHub Pages

## ⚙️ Kustomisasi

### Mengubah Nomor WhatsApp
Ganti semua instance `6285163689179` di:
- `index.html` (beberapa tempat)
- `script.js` (fungsi bookPackage dan form)

### Mengubah Harga
Edit di `index.html` pada:
- Section packages (baris ~200-300)
- Section pricing (baris ~600-700)

### Menambah Rute Baru
1. Tambah di section routes (`index.html`)
2. Tambah gambar di folder `gallery/`
3. Update script.js jika perlu
